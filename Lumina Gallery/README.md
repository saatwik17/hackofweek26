## 🚀 Quick Start (Local)

```bash
git clone https://github.com/<your-username>/<photography-repo>.git
cd flexible-magenta-gallery
npm install  # or open index.html directly
npm run dev  # if using dev server
```

Preview at `http://localhost:3000`.

---

## 📦 Image Optimization & Deployment

### Optimize Images
```bash
# Convert to WebP/AVIF (if using ImageMagick)
npm run optimize-images
```

### Deploy to EdgeOne Pages
1. 📁 Upload `dist/` or root folder to Pages  
2. ⚙️ Enable **Image CDN** for automatic optimization  
3. 🚀 Deploy – images served from 1000+ edge nodes worldwide
