# 📇 Nexus Directory

A futuristic contact directory app built with **React 19**, **TypeScript**, **Motion**, and **Tailwind CSS**.  
This project combines **binary search**, **smart filtering**, and a premium 3D UI to create a fast and modern contact management experience.

[🚀 Live Demo](https://guilty-rose-uzs6tyjkfn.edgeone.app/)

---

## ✨ Overview

Nexus Directory is a visually rich contact exploration app that lets users search through a large contact list using **prefix-based binary search**.  
Contacts are organized into categories like Work, Personal, Family, and VIP, while a floating 3D background and tilt-card layout make the interface feel modern and immersive.

The project focuses on:
- Fast search performance.
- Smooth filtering by category.
- Animated UI interactions.
- Elegant visual presentation.

---

## 🔥 Key Features

- **🔍 Binary Search Contact Lookup**  
  Search contacts instantly using prefix matching powered by binary search.

- **🏷️ Category Filters**  
  Browse contacts by Work, Personal, Family, and VIP tabs.

- **🎴 3D Tilt Cards**  
  Each contact card has a premium hover effect with depth and motion.

- **🌌 Floating Background Visuals**  
  Animated image layers create a futuristic atmosphere.

- **⚡ Performance Stats**  
  Displays real search time and match count to highlight efficiency.

- **📱 Responsive Layout**  
  Works smoothly across desktop, tablet, and mobile screens.

## 🧠 Core Logic

This project uses a **binary search prefix matching algorithm** to find contacts quickly in a sorted list.  
That means the search is much faster than scanning every contact one by one.

It also includes:
- Alphabetically sorted generated data.
- Dynamic category switching.
- Highlighted results when searching.
- A clean empty state when no matches are found.

---

## 🚀 How It Works

1. The contact list is generated and sorted alphabetically.
2. When a user types in the search bar, binary search finds matching prefixes.
3. The app updates results instantly.
4. Users can also switch categories to narrow down the list.
5. Results are displayed inside animated 3D cards.
