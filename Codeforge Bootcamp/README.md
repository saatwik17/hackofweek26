## ✨ Bootcamp Features

- 📚 **Structured Curriculum** covering HTML, CSS, JS, React, Node.js  
- 💻 **Hands-on Projects** with real-world applications  
- 👥 **Mentor Support** from industry professionals

## 🛠️ Tech Stack

- 🌐 **Frontend:** React.js, Tailwind CSS, Framer Motion  
- ⚛️ **State Management:** Context API / Redux  
- 📊 **Data Visualization:** Chart.js / Recharts  
- ☁️ **Hosting:** Tencent **EdgeOne Pages** (global CDN + edge delivery)  
- 🔧 **Build Tools:** Vite / Webpack
