## рџ§  Core Logic

The main highlight of this project is its **undo/redo architecture**.  
Every user action like add, remove, edit, or toggle is stored as an action object in a history stack, which allows the app to move backward or forward through task changes.

This makes the project a strong showcase of:
- State management.
- Action-based architecture.
- UI responsiveness.
- Real-time interaction handling.

## рџљЂ How It Works

1. Add a task using the input field.
2. The task appears in the list with animation.
