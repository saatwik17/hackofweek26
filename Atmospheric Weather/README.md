## 📦 Build & Deployment

### Build

```bash
# Example for a typical JS framework
npm run build
```

This generates an optimized production build in a directory like `dist` or `build` (configure according to your tooling).

### Deploy to EdgeOne Pages

You can deploy using Git-based workflows or drag-and-drop uploads:

1. 🔑 Log in to your **Tencent Cloud / EdgeOne Pages** console.  
2. 📂 Create a Pages project: connect your Git repo, start from a template, or upload static files.  
3. ⚙️ Configure:
   - Build command (e.g., `npm run build`)  
   - Output directory (e.g., `dist` or `build`)  
4. 🌍 Deploy and let the edge network distribute your site globally.  

For quick static sites, you can also use a drag‑and‑drop interface (Pages Drop) to upload built files and go live in seconds.
