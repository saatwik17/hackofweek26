# 🧠 Mindspark Quiz – AI Quiz Generator

**Preliminary Teal** is your ultimate **AI-powered Quiz Generator** that creates unlimited customized quizzes, flashcards, and assessments instantly. 🎯  
Hosted on **EdgeOne Pages** for lightning-fast global access.

🔗 **Live Demo:** [Click here to generate quizzes](https://preliminary-teal-dhkfyjibdt.edgeone.app) ⚡

---

## ✨ AI Quiz Features

- 🤖 **Instant Quiz Generation** from any topic/subject  
- 📝 **Multiple Formats**: MCQ, True/False, Fill-in-Blanks, Short Answer  
- 🎯 **Difficulty Levels**: Easy, Medium, Hard, Expert  
- 📊 **Auto-grading** with detailed explanations  
- 🔄 **Regenerate & Customize** until perfect  
- 📱 **Mobile-first** responsive interface  
- 💾 **Save & Share** your quizzes easily  

---

## 🛠️ Tech Stack

- 🤖 **AI Core:** OpenAI GPT / Custom LLM integration  
- ⚛️ **Frontend:** React.js + Tailwind CSS  
- ⚙️ **State Management:** Zustand / Redux Toolkit  
- 📱 **UI Components:** Headless UI / Radix UI  
- 📊 **Charts:** Recharts for analytics  
- ☁️ **Hosting:** Tencent **EdgeOne Pages** (edge functions + CDN)  

---

## 🚀 Quick Start

```bash
git clone https://github.com/<your-username>/ai-quiz-generator.git
cd preliminary-teal-quiz
npm install
npm run dev
```

Open `http://localhost:3000` and start generating quizzes!

---

## 📦 Deployment Guide

### Environment Setup
```bash
# Add your API keys
cp .env.example .env.local
# OPENAI_API_KEY=your-key-here
```

### Production Build & Deploy
```bash
npm run build
# Deploy dist/ to EdgeOne Pages
# Enable Edge Functions for AI API calls
```
