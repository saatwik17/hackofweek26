## рџљЂ How It Works

1. Add a new job to the queue.
2. The job enters the pending list.
3. The printer picks it up automatically.
4. Printing is simulated based on the number of pages.
5. The job is moved to the completed section.
6. The cycle continues with smooth animated transitions.
