# ✅ To-Do-List App – Smart To-Do List

**Preliminary Teal** is your ultimate **productive To-Do List** app with drag-and-drop tasks, smart prioritization, and seamless cross-device sync. 🚀  
Hosted on **EdgeOne Pages** for instant global access.

🔗 **Live Demo:** [Click here to manage your tasks](https://preliminary-teal-dhkfyjibdt.edgeone.app/) 📱

---

## ✨ To-Do Features

- ✅ **Drag & Drop** task reordering  
- 🎨 **Categories & Color Coding** (Work, Personal, Urgent)  
- ⏰ **Due Dates & Reminders**  
- 📱 **Mobile-first Responsive** design  
- 💾 **Local Storage** – never lose your tasks  
- 🔍 **Smart Search & Filter** completed/pending  
- 📊 **Progress Dashboard** with completion stats  

---

## 🛠️ Tech Stack

- ⚛️ **Frontend:** React.js + Vite  
- 🎨 **Styling:** Tailwind CSS + Custom Animations  
- 🗂️ **State:** Zustand for local persistence  
- ✨ **UI:** Framer Motion for smooth drag-drop  
- 📊 **Charts:** Chart.js for productivity stats  
- ☁️ **Hosting:** Tencent **EdgeOne Pages** (global CDN)  

---

## 🚀 Quick Start

```bash
git clone https://github.com/<your-username>/todo-app.git
cd preliminary-teal-todo
npm install
npm run dev
```

Open `http://localhost:5173` and add your first task!

---

## 📦 Deployment

### Build & Deploy
```bash
npm run build
# Upload dist/ folder to EdgeOne Pages
# No server needed – 100% client-side!
```
