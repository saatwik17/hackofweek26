💳 # Credit Card Recommendation AI Agent

> An intelligent AI-powered web app that recommends the best credit card based on user spending habits and financial profile.  

🌐 **Live Demo:** 👉 [Click Here](https://acceptable-brown-o2pdvwoxap.edgeone.app/)


***

## 🎯 Overview  

This project was built for the **Hack-N-Walk Hackathon (Hack-O-Week)** and solves the problem of helping users choose the most suitable credit card using an AI-based recommendation approach.  
The system takes user inputs about spending patterns and basic financial details, then suggests credit cards that match their needs and benefits.  

***

## ✨ Features  

- 🤖 AI-inspired credit card recommendations  
- 💰 Considers spending habits and financial profile  
- 🖥️ Simple and clean user interface  
- 📱 Responsive design for desktop and mobile
