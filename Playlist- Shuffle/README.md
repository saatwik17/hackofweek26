## рџ§  Core Logic

The main data structure behind this project is a **doubly linked list**, which allows traversal and updates in both forward and backward directions.  
This makes it suitable for playlist-like systems where tracks can move, reverse, or be inserted from either side.

The project demonstrates:
- Doubly linked list operations.
- Dynamic rendering of node relationships.
- React state synchronization with a custom class-based structure.
- Interactive visualization of abstract concepts.
