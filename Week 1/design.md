# Portfolio Website Design Style

## Design Philosophy

### Color Palette
- **Primary**: Deep Navy (#1a2332) - Professional, trustworthy, academic
- **Secondary**: Warm White (#fafafa) - Clean, modern, spacious  
- **Accent**: Soft Gold (#d4af37) - Achievement, excellence, premium feel
- **Supporting**: Muted Blue-Gray (#6b7280) - Sophisticated, readable

### Typography
- **Display Font**: "Playfair Display" - Elegant serif for headings, conveys academic gravitas
- **Body Font**: "Inter" - Clean, modern sans-serif for excellent readability
- **Code Font**: "JetBrains Mono" - Technical precision for skill demonstrations

### Visual Language
- **Editorial Sophistication**: Inspired by high-end tech publications like Wired and The Gentlewoman
- **Minimalist Precision**: Clean layouts with purposeful white space
- **Subtle Luxury**: Refined details without ostentation
- **Academic Excellence**: Professional presentation suitable for college and company review

## Visual Effects

### Used Libraries
- **Anime.js**: Smooth micro-interactions and element animations
- **ECharts.js**: Interactive skills radar chart and data visualizations
- **Typed.js**: Typewriter effect for dynamic text in hero section
- **Splide.js**: Elegant project carousel and image galleries
- **p5.js**: Subtle particle background effects for visual interest
- **Splitting.js**: Advanced text animations and letter-by-letter effects

### Animation Strategy
- **Hero Typewriter**: Dynamic introduction text with cursor effect
- **Skills Radar**: Animated chart revealing proficiency levels on scroll
- **Project Cards**: Subtle hover transformations with depth and shadow
- **Timeline**: Smooth reveal animations for experience points
- **Background**: Gentle particle system creating ambient movement

### Styling Approach
- **Grid-Based Layout**: Precise alignment and responsive structure
- **Layered Depth**: Subtle shadows and elevation for visual hierarchy
- **Smooth Transitions**: 300ms easing for all interactive elements
- **Focus States**: Clear accessibility indicators for keyboard navigation

### Header Effect
- **Clean Navigation**: Fixed header with smooth scroll-to-section behavior
- **Logo Animation**: Subtle hover effect on portfolio branding
- **Section Indicators**: Active state highlighting current page section

### Responsive Design
- **Mobile-First**: Optimized for high-resolution displays and mobile devices
- **Flexible Typography**: Scalable text maintaining readability across devices
- **Touch-Friendly**: Adequate spacing for mobile interaction
- **Performance**: Optimized animations that respect user motion preferences
